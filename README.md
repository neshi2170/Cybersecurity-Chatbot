# 🛡️ Cybersecurity Awareness Chatbot

## Project Overview
An interactive cybersecurity awareness chatbot that helps users learn about online safety through conversation. The chatbot features voice greetings, memory capabilities, sentiment detection, and a modern GUI interface.

## Features Implemented

### Part 1 Features
- ✅ Voice greeting with WAV file playback
- ✅ ASCII art logo display
- ✅ Personalized user interaction
- ✅ Input validation and error handling
- ✅ Enhanced console UI with colors

### Part 2 Features
- ✅ **WPF GUI Interface** - Modern dark theme design with chat bubbles
- ✅ **Keyword Recognition** - Detects: passwords, scams, privacy, phishing, malware, breaches
- ✅ **Random Responses** - Multiple tips per topic, randomly selected
- ✅ **Conversation Flow** - Handles follow-ups like "tell me more", "another tip"
- ✅ **Memory System** - Remembers user's name and favorite cybersecurity topics
- ✅ **Sentiment Detection** - Detects worried, curious, frustrated, grateful moods
- ✅ **Error Handling** - Graceful handling of unrecognized input
- ✅ **Code Optimization** - Uses dictionaries, lists, and OOP principles
